# Planck'd 2025 Submission: Quantum Machine Learning Track

This repository contains classical and quantum-hybrid baselines for MNIST classification.

Contents
- code/: training and evaluation scripts for CNN, SVM, and a PennyLane-Torch hybrid VQA classifier
- results/: saved metrics, plots, and model artifacts
- report.pdf: autogenerated summary report

Quick start (Windows PowerShell)
1. python -m venv .venv
2. .venv\\Scripts\\Activate.ps1
3. python -m pip install --upgrade pip
4. pip install -r code/requirements.txt

Run baselines
- CNN: python code/train.py --task mnist --model cnn --epochs 3 --batch-size 128 --out results/cnn
- SVM: python code/train.py --task mnist --model svm --out results/svm
- Hybrid VQA: python code/train.py --task mnist --model vqa --epochs 3 --batch-size 128 --out results/vqa

Evaluate and create report
- python code/eval.py --runs results/cnn results/svm results/vqa --out results/summary
- python code/report/generate_report.py --summary results/summary/metrics_summary.json --out report.pdf

Reproducibility
- All scripts accept seeds and device flags
- Set --subset-train and --subset-test for quick runs; remove to run full MNIST
